## What is DevEfficiency?

DevEfficiency is a full-featured service request management platform that streamlines the workflow from client onboarding through service delivery, billing, and financial reporting. Built with modern web technologies, it provides developers with a robust system for managing client relationships, project workflows, and business operations.

## Key Features

<AccordionGroup>
  <Accordion title="Multi-Status Service Orders" icon="workflow">
    Service orders progress through customizable status workflows: Em avaliação, Orçamento enviado, Na fila, Em correções, Pendente, Finalizado, and Cancelado. Each status transition can trigger automated actions and notifications.
  </Accordion>
  
  <Accordion title="Protocol System" icon="file-contract">
    Protocols link service orders to billing cycles with support for multiple products, time-based registers, and receipts. Track protocol status from Em aberto through Liberado para pagamento to Fechado or Cancelado.
  </Accordion>
  
  <Accordion title="PDF Generation" icon="file-pdf">
    Automatically generate professional PDF documents for service orders, billing statements, and protocols using Puppeteer with customizable EJS templates.
  </Accordion>
  
  <Accordion title="Financial Management" icon="money-bill-wave">
    Complete financial tracking with bank accounts, payables, cost centers, bank transfers, and comprehensive reporting including DRE (Demonstração do Resultado do Exercício) and expense reports.
  </Accordion>
  
  <Accordion title="JWT Authentication" icon="shield-halved">
    Secure JWT-based authentication with token refresh capabilities, rate limiting, and security headers via Fastify middleware.
  </Accordion>
  
  <Accordion title="Email Notifications" icon="envelope">
    Automated email notifications powered by Nodemailer for important events like status changes, billing reminders, and protocol updates.
  </Accordion>
</AccordionGroup>

## Technology Stack

DevEfficiency is built on a modern, production-ready technology stack:

- **Backend**: Fastify (Node.js/TypeScript) with rate limiting and security headers
- **Database**: MySQL 8.4 with Sequelize ORM and TypeScript decorators
- **Authentication**: JWT tokens with bcrypt password hashing
- **PDF Generation**: Puppeteer with EJS templating
- **Email**: Nodemailer for SMTP-based notifications
- **Frontend**: Server-side rendered HTML/EJS with static assets
- **Deployment**: Docker and Docker Compose support

<Note>
DevEfficiency requires Node.js 18+ and MySQL 8.4+ for optimal compatibility.
</Note>
