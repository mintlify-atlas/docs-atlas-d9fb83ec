---
title: "List Projects"
api: "GET /api/projects"
description: "Retrieve a list of all projects with optional filtering"
---

## Authentication

This endpoint requires JWT authentication. Include the token in the Authorization header.

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

## Query Parameters

<ParamField query="filter" type="string" optional>
  Filter projects by specific criteria.
  
  **Available values:**
  - `fixed` - Returns only pinned/fixed projects
  
  When omitted, returns all projects.
</ParamField>

## Response

Returns an array of project objects with associated client information.

<ResponseField name="id" type="number">
  Unique identifier for the project
</ResponseField>

<ResponseField name="name" type="string">
  Project name (max 128 characters)
</ResponseField>

<ResponseField name="url" type="string" optional>
  Project URL or endpoint
</ResponseField>

<ResponseField name="fixed" type="boolean" default="false">
  Indicates if the project is pinned/fixed for quick access
</ResponseField>

<ResponseField name="type" type="enum" optional>
  Project type classification
  
  **Possible values:**
  - `API` - API integration project
  - `Bot` - Bot or automation script
  - `WebSite` - Website or web application
  - `Automação` - General automation project
  - `Crawler` - Web scraping or crawler project
  - `Outros` - Other project types
</ResponseField>

<ResponseField name="ClientId" type="number">
  Foreign key reference to the associated client
</ResponseField>

<ResponseField name="ServerId" type="number" optional>
  Foreign key reference to the server hosting this project
</ResponseField>

<ResponseField name="Client" type="object">
  Associated client information
  
  <Expandable title="Client object properties">
    <ResponseField name="id" type="number">
      Client unique identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Client name
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      Client creation timestamp
    </ResponseField>
    
    <ResponseField name="updatedAt" type="string">
      Client last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string">
  Project creation timestamp (ISO 8601 format)
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Project last update timestamp (ISO 8601 format)
</ResponseField>

## Examples

### List All Projects

<CodeGroup>
```bash cURL
curl -X GET "https://api.devefficiency.com/api/projects" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/projects', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  }
});

const projects = await response.json();
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.devefficiency.com/api/projects',
    headers=headers
)

projects = response.json()
```
</CodeGroup>

### List Fixed Projects Only

<CodeGroup>
```bash cURL
curl -X GET "https://api.devefficiency.com/api/projects?filter=fixed" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/projects?filter=fixed', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  }
});

const fixedProjects = await response.json();
```

```python Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
}

response = requests.get(
    'https://api.devefficiency.com/api/projects',
    headers=headers,
    params={'filter': 'fixed'}
)

fixed_projects = response.json()
```
</CodeGroup>
