---
title: "Get Project"
api: "GET /api/projects/:id"
description: "Retrieve detailed information about a specific project including all relationships"
---

## Authentication

This endpoint requires JWT authentication. Include the token in the Authorization header.

```bash
Authorization: Bearer YOUR_JWT_TOKEN
```

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the project to retrieve
</ParamField>

## Response

Returns a detailed project object with all associated relationships including subprojects, service orders, checks, subscriptions, and client information.

<ResponseField name="id" type="number">
  Unique identifier for the project
</ResponseField>

<ResponseField name="name" type="string">
  Project name (max 128 characters)
</ResponseField>

<ResponseField name="url" type="string" optional>
  Project URL or endpoint
</ResponseField>

<ResponseField name="fixed" type="boolean" default="false">
  Indicates if the project is pinned/fixed for quick access. Fixed projects appear prominently in the UI for easy access to frequently used projects.
</ResponseField>

<ResponseField name="type" type="enum" optional>
  Project type classification
  
  **Possible values:**
  - `API` - API integration project
  - `Bot` - Bot or automation script
  - `WebSite` - Website or web application
  - `Automação` - General automation project
  - `Crawler` - Web scraping or crawler project
  - `Outros` - Other project types
</ResponseField>

<ResponseField name="ClientId" type="number">
  Foreign key reference to the associated client
</ResponseField>

<ResponseField name="ServerId" type="number" optional>
  Foreign key reference to the server hosting this project
</ResponseField>

<ResponseField name="Client" type="object">
  Associated client information
  
  <Expandable title="Client object properties">
    <ResponseField name="id" type="number">
      Client unique identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Client name
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      Client creation timestamp
    </ResponseField>
    
    <ResponseField name="updatedAt" type="string">
      Client last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="Subprojects" type="array">

  <Expandable title="Subproject object properties">
    <ResponseField name="id" type="number">
      Subproject unique identifier
    </ResponseField>
    
    <ResponseField name="name" type="string">
      Subproject name
    </ResponseField>
    
    <ResponseField name="ProjectId" type="number">
      Reference to parent project
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      Subproject creation timestamp
    </ResponseField>
    
    <ResponseField name="updatedAt" type="string">
      Subproject last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="Service_orders" type="array">
  Array of service orders associated with this project
  
  <Expandable title="Service order object properties">
    <ResponseField name="id" type="number">
      Service order unique identifier
    </ResponseField>
    
    <ResponseField name="ProjectId" type="number">
      Reference to associated project

    <ResponseField name="createdAt" type="string">

<ResponseField name="Subscription" type="array">
  Array of subscriptions associated with this project
  
  <Expandable title="Subscription object properties">

    <ResponseField name="createdAt" type="string">

    </ResponseField>
    
    <ResponseField name="updatedAt" type="string">
      Subscription last update timestamp

  </Expandable>

<ResponseField name="createdAt" type="string">

<ResponseField name="updatedAt" type="string">

## Examples

<CodeGroup>
```bash cURL
curl -X GET "https://api.devefficiency.com/api/projects/1" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json"
```

```javascript JavaScript
const projectId = 1;

const response = await fetch(`https://api.devefficiency.com/api/projects/${projectId}`, {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  }
});

const project = await response.json();
```

```python Python
import requests

project_id = 1

headers = {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
}

response = requests.get(
    f'https://api.devefficiency.com/api/projects/{project_id}',
    headers=headers
)

project = response.json()
```
</CodeGroup>

## Error Responses

<ResponseField name="401 Unauthorized" type="object">
  Authentication token is missing or invalid
  
  ```json
  {
    "statusCode": 401,
    "error": "Unauthorized",
    "message": "Missing or invalid authentication token"
  }
  ```
</ResponseField>

<ResponseField name="404 Not Found" type="object">
  Project with the specified ID does not exist
  
  ```json
  {
    "statusCode": 404,
    "error": "Not Found",
    "message": "Project not found"
  }
  ```
</ResponseField>

<ResponseField name="500 Internal Server Error" type="object">
  Server encountered an unexpected error
  
  ```json
  {
    "statusCode": 500,
    "error": "Internal Server Error",
    "message": "An unexpected error occurred"
  }
  ```
</ResponseField>
