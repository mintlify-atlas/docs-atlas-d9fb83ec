---
title: 'Get Contact'
api: 'GET /api/contacts/:id'
description: 'Retrieve a specific contact by ID'
---

## Authentication

This endpoint requires JWT authentication.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the contact
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the contact
</ResponseField>

<ResponseField name="name" type="string">
  Name of the contact (max 128 characters)
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the contact (optional)
</ResponseField>

<ResponseField name="number" type="string">
  Phone number of the contact (optional)
</ResponseField>

<ResponseField name="whatsapp" type="string">
  WhatsApp number of the contact (optional)
</ResponseField>

<ResponseField name="ClientId" type="number">
  Foreign key reference to the associated client
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the contact was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the contact was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.devefficiency.com/api/contacts/1 \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```
</RequestExample>
