The Bank Accounts API allows you to manage bank accounts within the DevEfficiency platform. Bank accounts are used to track balances, process payables, and manage transfers between accounts.

## The Bank Account Object

Bank accounts represent financial accounts where your organization's money is held.

### Attributes

<ParamField path="id" type="number">
  Unique identifier for the bank account
</ParamField>

<ParamField path="name" type="string" required>
  Display name for the bank account (max 128 characters)
</ParamField>

<ParamField path="bank" type="string">
  Name of the banking institution (max 128 characters)
</ParamField>

<ParamField path="agency" type="string">
  Bank branch or agency number (max 20 characters)
</ParamField>

<ParamField path="accountNumber" type="string">
  Account number at the bank (max 20 characters)
</ParamField>

<ParamField path="balance" type="number">
  Current account balance. Defaults to 0.0. This field is automatically updated by transactions and cannot be modified directly via the update endpoint.
</ParamField>

<ParamField path="createdAt" type="string">
  Timestamp when the bank account was created (ISO 8601 format)
</ParamField>

<ParamField path="updatedAt" type="string">
  Timestamp when the bank account was last updated (ISO 8601 format)
</ParamField>

## Create a Bank Account

Create a new bank account in the system.

<CodeGroup>
```bash cURL
curl --request POST \
  --url https://api.devefficiency.com/api/bank-accounts \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Payroll Account",
    "bank": "Chase Bank",
    "agency": "9012",
    "accountNumber": "555666777",
    "balance": 25000.00
  }'
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/bank-accounts', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Payroll Account',
    bank: 'Chase Bank',
    agency: '9012',
    accountNumber: '555666777',
    balance: 25000.00
  })
});
const newAccount = await response.json();
```

```python Python
import requests

response = requests.post(
  'https://api.devefficiency.com/api/bank-accounts',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  json={
    'name': 'Payroll Account',
    'bank': 'Chase Bank',
    'agency': '9012',
    'accountNumber': '555666777',
    'balance': 25000.00
  }
)
new_account = response.json()
```
</CodeGroup>

## Update a Bank Account

Update an existing bank account's information. Note that the balance cannot be updated directly through this endpoint for security reasons. Balances are automatically updated through payables and bank transfers.

<CodeGroup>
```bash cURL
curl --request PUT \
  --url https://api.devefficiency.com/api/bank-accounts/3 \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Updated Payroll Account",
    "bank": "Chase Bank",
    "agency": "9012",
    "accountNumber": "555666888"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/bank-accounts/3', {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Updated Payroll Account',
    bank: 'Chase Bank',
    agency: '9012',
    accountNumber: '555666888'
  })
});
const updatedAccount = await response.json();
```

```python Python
import requests

response = requests.put(
  'https://api.devefficiency.com/api/bank-accounts/3',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  json={
    'name': 'Updated Payroll Account',
    'bank': 'Chase Bank',
    'agency': '9012',
    'accountNumber': '555666888'
  }
)
updated_account = response.json()
```
</CodeGroup>

### Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the bank account to update
</ParamField>

### Request Body

<ParamField body="name" type="string">
  Display name for the bank account (max 128 characters)
</ParamField>

<ParamField body="bank" type="string">
  Name of the banking institution (max 128 characters)
</ParamField>

<ParamField body="agency" type="string">

## Delete a Bank Account

Permanently delete a bank account from the system.

<Warning>
  This action is irreversible. Ensure the account has no pending transactions before deletion.
</Warning>

<CodeGroup>
```bash cURL
curl --request DELETE \
  --url https://api.devefficiency.com/api/bank-accounts/3 \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/bank-accounts/3', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});
```

```python Python
import requests

response = requests.delete(
  'https://api.devefficiency.com/api/bank-accounts/3',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
)
```
</CodeGroup>

### Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the bank account to delete
</ParamField>
