The Financial Reports API provides powerful reporting endpoints for analyzing financial data, including transaction history, income statements (DRE), and expense analysis. These reports help you understand cash flow, profitability, and spending patterns.

## Financial History

Retrieve a comprehensive transaction history showing all cash inflows and outflows, including receipts, payables, and bank transfers.

<CodeGroup>
```bash cURL
curl --request GET \
  --url 'https://api.devefficiency.com/api/financial-history?startDate=2024-03-01&endDate=2024-03-31' \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json'
```

```javascript JavaScript
const params = new URLSearchParams({
  startDate: '2024-03-01',
  endDate: '2024-03-31'
});

const response = await fetch(
  `https://api.devefficiency.com/api/financial-history?${params}`,
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN',
      'Content-Type': 'application/json'
    }
  }
);
const history = await response.json();
```

```python Python
import requests

params = {
  'startDate': '2024-03-01',
  'endDate': '2024-03-31'
}

response = requests.get(
  'https://api.devefficiency.com/api/financial-history',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  params=params
)
history = response.json()
```
</CodeGroup>

### Query Parameters

<ParamField query="startDate" type="string">
  Start date for the report period (ISO 8601 format or YYYY-MM-DD). If omitted, returns all historical data.
</ParamField>

<ParamField query="endDate" type="string">
  End date for the report period (ISO 8601 format or YYYY-MM-DD). If omitted, returns all historical data.
</ParamField>

## DRE Report (Income Statement)

Generate a DRE (Demonstração do Resultado do Exercício) report, which is similar to a profit and loss statement. This report shows revenue, costs, expenses, and profitability for a specific period.

<CodeGroup>
```bash cURL
curl --request GET \
  --url 'https://api.devefficiency.com/api/dre-report?month=3&year=2024' \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json'
```

```javascript JavaScript
const params = new URLSearchParams({
  month: '3',
  year: '2024'
});

const response = await fetch(
  `https://api.devefficiency.com/api/dre-report?${params}`,
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN',
      'Content-Type': 'application/json'
    }
  }
);
const dreReport = await response.json();
```

```python Python
import requests

params = {
  'month': '3',
  'year': '2024'
}

response = requests.get(
  'https://api.devefficiency.com/api/dre-report',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  params=params
)
dre_report = response.json()
```
</CodeGroup>

### Query Parameters

<ParamField query="month" type="string">
  Month number (1-12). Defaults to current month if not specified.
</ParamField>

<ParamField query="year" type="string">
  Year (e.g., 2024). Defaults to current year if not specified.
</ParamField>

### Response

<ResponseField name="revenue" type="number">

</ResponseField>

<ResponseField name="directCosts" type="object">
  Cost of goods sold (CPV) breakdown
  <Expandable title="Direct Costs">
    <ResponseField name="productCosts" type="number">
      Total cost of products used in closed protocols
    </ResponseField>
    <ResponseField name="serviceExpenses" type="number">
      Total service expenses (protocol registers with type = 'Despesa')
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="operatingExpenses" type="object">
  Fixed operating expenses breakdown
  <Expandable title="Operating Expenses">
    <ResponseField name="total" type="number">
      Total operating expenses from payables
    </ResponseField>
    <ResponseField name="byCostCenter" type="array">
      Array of cost center expense breakdowns
      <Expandable title="Cost Center Breakdown">
        <ResponseField name="name" type="string">
          Cost center name
        </ResponseField>
        <ResponseField name="value" type="number">
          Total expenses for this cost center
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>

<ResponseField name="grossProfit" type="number">
  Gross profit (revenue - direct costs)
</ResponseField>

### DRE Calculation Logic

<Tabs>
  <Tab title="Revenue">
    Revenue is calculated from all receipts associated with protocols that were **closed** during the period.
    
    - Filters protocols by `closedAt` date within the month/year
    - Only includes protocols with `status = 'Fechado'`
    - Sums all receipt values linked to these protocols
  </Tab>
  <Tab title="Direct Costs">
    Direct costs include:
    
    **Product Costs:**
    - Products used in protocols closed during the period
    - Based on product cost (COGS) from the Product model
    
    **Service Expenses:**
    - Protocol register entries with `type = 'Despesa'`
    - Only from protocols closed in the period
  </Tab>
  <Tab title="Operating Expenses">
    Operating expenses are fixed costs from payables:
    
    - Payables **created** during the period (by `createdAt`)
    - Excludes cancelled payables (`status != 'cancelado'`)
    - Grouped and summed by cost center
    - Payables without a cost center are grouped under "Geral"
  </Tab>
  <Tab title="Profitability">
    **Gross Profit** = Revenue - (Product Costs + Service Expenses)
    
    **Net Result** = Gross Profit - Operating Expenses
    
    A positive net result indicates profit, while negative indicates a loss for the period.
  </Tab>
</Tabs>

## Expense Report

Generate a hierarchical expense report showing paid expenses organized by cost center structure. This report rolls up expenses from child cost centers to parents, providing a complete organizational view.

<CodeGroup>
```bash cURL
curl --request GET \
  --url 'https://api.devefficiency.com/api/expense-report?startDate=2024-03-01&endDate=2024-03-31' \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json'
```

```javascript JavaScript
const params = new URLSearchParams({
  startDate: '2024-03-01',
  endDate: '2024-03-31'
});

const response = await fetch(
  `https://api.devefficiency.com/api/expense-report?${params}`,
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer YOUR_JWT_TOKEN',
      'Content-Type': 'application/json'
    }
  }
);
const expenseReport = await response.json();
```

```python Python
import requests

params = {
  'startDate': '2024-03-01',
```

### Roll-up Calculation

The expense report uses a bottom-up aggregation approach:

1. **Direct Expenses**: Each cost center's `total` initially includes only payables directly assigned to it
2. **Child Aggregation**: The totals from all child cost centers are recursively summed
3. **Parent Roll-up**: Each parent's total includes its own direct expenses plus all children's totals
4. **Tree Structure**: Only root-level cost centers (ParentId = null) are returned at the top level

**Example:**
- Office Supplies (leaf): $3,250.50 in direct expenses
- Travel (leaf): $15,499.50 in direct expenses
- Administrative (parent): $0 direct + $3,250.50 + $15,499.50 = $18,750.00 total
- Facilities (leaf): $13,750.00 in direct expenses
- Operations (root): $0 direct + $18,750.00 + $13,750.00 = $32,500.00 total

<Info>
  The expense report only includes **paid** payables (status = 'pago') filtered by payment date. Pending or cancelled payables are excluded from the report.
</Info>
