The Cost Centers API allows you to organize and categorize expenses in a hierarchical structure. Cost centers support parent-child relationships, enabling multi-level expense tracking and reporting.

## The Cost Center Object

Cost centers represent expense categories used for financial organization and reporting.

### Attributes

<ParamField path="id" type="number">
  Unique identifier for the cost center
</ParamField>

<ParamField path="name" type="string" required>
  Display name for the cost center (max 128 characters)
</ParamField>

<ParamField path="description" type="string">
  Optional description of the cost center's purpose
</ParamField>

<ParamField path="ParentId" type="number">
  ID of the parent cost center. Null for root-level cost centers. This enables hierarchical organization.
</ParamField>

<ParamField path="createdAt" type="string">
  Timestamp when the cost center was created (ISO 8601 format)
</ParamField>

<ParamField path="updatedAt" type="string">
  Timestamp when the cost center was last updated (ISO 8601 format)
</ParamField>

<ParamField path="parent" type="object">
  Populated parent cost center object when included in query
</ParamField>

<ParamField path="children" type="array">
  Array of child cost center objects when included in query. Can be nested up to 5 levels deep.
</ParamField>

## List All Cost Centers

Retrieve all cost centers in a hierarchical tree structure. The response includes root-level cost centers with their nested children (up to 5 levels deep).

<CodeGroup>
```bash cURL
curl --request GET \
  --url https://api.devefficiency.com/api/cost-centers \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json'
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/cost-centers', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  }
});
const costCenters = await response.json();
```

```python Python
import requests

response = requests.get(
  'https://api.devefficiency.com/api/cost-centers',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  }
)
cost_centers = response.json()
```
</CodeGroup>

## Create a Cost Center

Create a new cost center. You can create either a root-level cost center (by omitting `ParentId`) or a child cost center (by providing a `ParentId`).

<CodeGroup>
```bash cURL
curl --request POST \
  --url https://api.devefficiency.com/api/cost-centers \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Marketing",
    "description": "Marketing and advertising expenses",
    "ParentId": null
  }'
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/cost-centers', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Marketing',
    description: 'Marketing and advertising expenses',
    ParentId: null
  })
});
const newCostCenter = await response.json();
```

```python Python
import requests

response = requests.post(
  'https://api.devefficiency.com/api/cost-centers',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN',
    'Content-Type': 'application/json'
  },
  json={
    'name': 'Marketing',
    'description': 'Marketing and advertising expenses',
    'ParentId': None
  }
)
new_cost_center = response.json()
```
</CodeGroup>

<CodeGroup>
```bash cURL

  --url https://api.devefficiency.com/api/cost-centers/8 \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \

    "name": "Marketing & Sales",
```

## Delete a Cost Center

Permanently delete a cost center from the system. Child cost centers will be automatically deleted due to the cascade delete behavior.

<Warning>
  This action is irreversible and will cascade delete all child cost centers. Ensure no payables are currently assigned to this cost center or its children before deletion.
</Warning>

<CodeGroup>
```bash cURL
curl --request DELETE \
  --url https://api.devefficiency.com/api/cost-centers/8 \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/api/cost-centers/8', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});
```

```python Python
import requests

response = requests.delete(
  'https://api.devefficiency.com/api/cost-centers/8',
  headers={
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
)
```
</CodeGroup>

### Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the cost center to delete
</ParamField>

### Response

<ResponseExample>

```json 404 - Not Found
}
```

## Hierarchical Structure

Cost centers support up to 5 levels of nesting, allowing you to create sophisticated organizational structures:

```
Operations (Level 1)
└── Administrative (Level 2)
    ├── Office Supplies (Level 3)
    │   └── Stationery (Level 4)
    │       └── Paper Products (Level 5)
    └── Travel (Level 3)
        ├── Domestic (Level 4)
        └── International (Level 4)
```

This hierarchical structure is particularly useful for:

- **Roll-up reporting**: Expenses aggregate from child to parent cost centers
- **Departmental organization**: Mirror your company structure
- **Budget allocation**: Distribute budgets across organizational units
- **Expense analysis**: Drill down from high-level to detailed categories
