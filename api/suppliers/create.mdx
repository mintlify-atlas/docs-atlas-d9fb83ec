---
title: 'Create Supplier'
api: 'POST /api/suppliers'
description: 'Create a new supplier'
---

## Authentication

This endpoint requires JWT authentication.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Body Parameters

<ParamField body="name" type="string" required>
  Name of the supplier (max 128 characters)
</ParamField>

<ParamField body="cnpj" type="string">
  Brazilian company tax ID (CNPJ) - max 20 characters, must be unique (optional)
</ParamField>

<ParamField body="email" type="string">
  Email address of the supplier - max 128 characters (optional)
</ParamField>

<ParamField body="phone" type="string">
  Phone number of the supplier - max 20 characters (optional)
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the created supplier
</ResponseField>

<ResponseField name="name" type="string">
  Name of the supplier
</ResponseField>

<ResponseField name="cnpj" type="string">
  Brazilian company tax ID (CNPJ)
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the supplier
</ResponseField>

<ResponseField name="phone" type="string">
  Phone number of the supplier
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the supplier was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the supplier was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.devefficiency.com/api/suppliers \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "New Supplier Ltd",
    "cnpj": "11.222.333/0001-44",
    "email": "info@newsupplier.com",
    "phone": "+5511777777777"
  }'
```
</RequestExample>
