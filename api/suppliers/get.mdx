---
title: 'Get Supplier'
api: 'GET /api/suppliers/:id'
description: 'Retrieve a specific supplier by ID'
---

## Authentication

This endpoint requires JWT authentication.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the supplier
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the supplier
</ResponseField>

<ResponseField name="name" type="string">
  Name of the supplier (max 128 characters)
</ResponseField>

<ResponseField name="cnpj" type="string">
  Brazilian company tax ID (CNPJ) - max 20 characters, unique (optional)
</ResponseField>

<ResponseField name="email" type="string">
  Email address of the supplier - max 128 characters (optional)
</ResponseField>

<ResponseField name="phone" type="string">
  Phone number of the supplier - max 20 characters (optional)
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the supplier was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the supplier was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.devefficiency.com/api/suppliers/1 \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```
</RequestExample>
