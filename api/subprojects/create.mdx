---
title: 'Create Subproject'
api: 'POST /api/subprojects'
description: 'Create a new subproject under a parent project'
---

## Authentication

This endpoint requires JWT authentication.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Body Parameters

<ParamField body="name" type="string" required>
  Name of the subproject (max 128 characters)
</ParamField>

<ParamField body="url" type="string" required>
  URL of the subproject repository or deployment
</ParamField>

<ParamField body="type" type="enum" required>
  Type of subproject. Possible values: `Front-End`, `Back-End`, `Bot`, `Outros`
</ParamField>

<ParamField body="ProjectId" type="number" required>
  Foreign key reference to the parent project
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the created subproject
</ResponseField>

<ResponseField name="name" type="string">
  Name of the subproject
</ResponseField>

<ResponseField name="url" type="string">
  URL of the subproject
</ResponseField>

<ResponseField name="type" type="enum">
  Type of subproject
</ResponseField>

<ResponseField name="ProjectId" type="number">
  Foreign key reference to the parent project
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the subproject was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the subproject was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl --request POST \
  --url https://api.devefficiency.com/api/subprojects \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Mobile App",
    "url": "https://github.com/company/mobile",
    "type": "Front-End",
    "ProjectId": 5
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 3,
  "name": "Mobile App",
  "url": "https://github.com/company/mobile",
  "type": "Front-End",
  "ProjectId": 5,
  "createdAt": "2024-03-08T15:45:00.000Z",
  "updatedAt": "2024-03-08T15:45:00.000Z"
}
```

```json 400
{
  "error": "Bad Request",
  "message": "Invalid subproject type. Must be one of: Front-End, Back-End, Bot, Outros"
}
```

```json 401
{
  "error": "Unauthorized",
  "message": "Invalid or missing authentication token"
}
```

```json 404
{
  "error": "Not Found",
  "message": "Project not found"
}
```

```json 500
{
  "error": "Internal Server Error"
}
```
</ResponseExample>
