---
title: 'Update Subproject'
api: 'PUT /api/subprojects/:id'
description: 'Update an existing subproject'
---

## Authentication

This endpoint requires JWT authentication.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Path Parameters

<ParamField path="id" type="number" required>
  The unique identifier of the subproject to update
</ParamField>

## Body Parameters

<ParamField body="name" type="string">
  Name of the subproject (max 128 characters)
</ParamField>

<ParamField body="url" type="string">
  URL of the subproject repository or deployment
</ParamField>

<ParamField body="type" type="enum">
  Type of subproject. Possible values: `Front-End`, `Back-End`, `Bot`, `Outros`
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the subproject
</ResponseField>

<ResponseField name="name" type="string">
  Updated name of the subproject
</ResponseField>

<ResponseField name="url" type="string">
  Updated URL of the subproject
</ResponseField>

<ResponseField name="type" type="enum">
  Updated type of subproject
</ResponseField>

<ResponseField name="ProjectId" type="number">
  Foreign key reference to the parent project
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the subproject was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the subproject was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl --request PUT \
  --url https://api.devefficiency.com/api/subprojects/1 \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Updated Frontend Application",
    "url": "https://github.com/company/frontend-v2",
    "type": "Front-End"
  }'
```
</RequestExample>

<ResponseExample>
```json 200
{
  "id": 1,
  "name": "Updated Frontend Application",
  "url": "https://github.com/company/frontend-v2",
  "type": "Front-End",
  "ProjectId": 5,
  "createdAt": "2024-01-15T10:30:00.000Z",
  "updatedAt": "2024-03-08T16:00:00.000Z"
}
```

```json 400
{
  "error": "Bad Request",
  "message": "Invalid subproject type"
}
```

```json 401
{
  "error": "Unauthorized",
  "message": "Invalid or missing authentication token"
}
```

```json 404
{
  "error": "Not Found",
  "message": "Subproject not found"
}
```

```json 500
{
  "error": "Internal Server Error"
}
```
</ResponseExample>
