---
title: 'List Subprojects'
api: 'GET /api/subprojects'
description: 'Retrieve a list of all subprojects'
---

## Authentication

This endpoint requires JWT authentication.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the subproject
</ResponseField>

<ResponseField name="name" type="string">
  Name of the subproject (max 128 characters)
</ResponseField>

<ResponseField name="url" type="string">
  URL of the subproject repository or deployment
</ResponseField>

<ResponseField name="type" type="enum">
  Type of subproject. Possible values: `Front-End`, `Back-End`, `Bot`, `Outros`
</ResponseField>

<ResponseField name="ProjectId" type="number">
  Foreign key reference to the parent project
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the subproject was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the subproject was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.devefficiency.com/api/subprojects \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```
</RequestExample>
