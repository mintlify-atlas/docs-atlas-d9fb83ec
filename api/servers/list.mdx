---
title: 'List Servers'
api: 'GET /api/servers'
description: 'Retrieve a list of all servers'
---

## Authentication

This endpoint requires JWT authentication.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the server
</ResponseField>

<ResponseField name="description" type="string">
  Description of the server (max 255 characters, optional)
</ResponseField>

<ResponseField name="provider" type="string">
  Cloud or hosting provider name (max 128 characters, optional)
</ResponseField>

<ResponseField name="host" type="string">
  Server hostname or IP address (max 128 characters, optional)
</ResponseField>

<ResponseField name="username" type="string">
  Username for server access (max 128 characters, optional)
</ResponseField>

<ResponseField name="password" type="string">
  Password for server access (max 128 characters, optional)
</ResponseField>

<ResponseField name="rsa" type="string">
  RSA private key for SSH access (text field, optional)
</ResponseField>

<ResponseField name="ClientId" type="number">
  Foreign key reference to the associated client (optional)
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the server was created
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Timestamp when the server was last updated
</ResponseField>

<RequestExample>
```bash cURL
curl --request GET \
  --url https://api.devefficiency.com/api/servers \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```
</RequestExample>
