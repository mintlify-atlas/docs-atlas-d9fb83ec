## Overview

This endpoint returns all products and services configured in DevEfficiency. Products represent billable items that can be associated with protocols and service orders. Each product includes pricing information, cost data, and license generation settings.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication. Include your JWT token in the format: `Bearer YOUR_JWT_TOKEN`
</ParamField>

## Response

<ResponseField name="products" type="array">
  Array of product objects
  
  <Expandable title="Product Object">
    <ResponseField name="id" type="integer">
      Unique identifier for the product
    </ResponseField>
    
    <ResponseField name="description" type="string" required>
      Product or service description. Maximum length: 255 characters
    </ResponseField>
    
    <ResponseField name="value" type="float" required>
      Sale price of the product. This is the amount charged to clients when the product is included in a protocol or service order. Defaults to 0 if not specified.
    </ResponseField>
    
    <ResponseField name="coust" type="float" required>
      Cost of the product to your organization. Used for profit margin calculations and financial reporting. Defaults to 0 if not specified.
    </ResponseField>
    
    <ResponseField name="generate_license" type="boolean" required>
      Indicates whether this product should trigger automatic license generation when added to a protocol. Set to `true` for software licenses or subscription-based services. Defaults to `false`.
    </ResponseField>
    
    <ResponseField name="createdAt" type="string">
      Timestamp when the product was created (ISO 8601 format)
    </ResponseField>
    
    <ResponseField name="updatedAt" type="string">
      Timestamp when the product was last updated (ISO 8601 format)
    </ResponseField>
  </Expandable>
</ResponseField>

## Example Request

```bash cURL
curl --request GET \
  --url https://api.devefficiency.com/products \
  --header 'Authorization: Bearer YOUR_JWT_TOKEN'
```

```javascript JavaScript
const response = await fetch('https://api.devefficiency.com/products', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  }
});

const products = await response.json();
```

## Usage in Protocols

Products retrieved from this endpoint can be associated with protocols through the Protocol Products API. When a product with `generate_license: true` is added to a protocol, the system automatically generates a license record. The product's `value` field can be used directly or overridden when creating protocol-product associations.

## Notes

- The response includes all products without pagination. For systems with large product catalogs, consider implementing filtering on the client side.
- Product pricing (`value` and `coust`) is stored as floating-point numbers to support decimal precision.
- The `generate_license` flag is critical for automated license management workflows.
